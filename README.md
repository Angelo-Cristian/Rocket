# 🎮 Rocket
Get on the rocket and reach the finish line while avoiding the dangers ahead of you. This is a 3D game made with Unity.
<img width="1615" height="862" alt="Rocket" src="https://github.com/user-attachments/assets/ead56e77-f503-498c-98d5-dac9ff7fd322" />

---

## 🛠️ Technical details
* **Movement system** -> Made with Unity's Input System: boost (SPACE), rotate left (A) / right (D).
* **Scene manager** -> There are 5 levels, each of them being represented by its own scene.
* **Finish line** -> When the rocket collides with the finish line, an animation plays, and after that, the next scene/level will be loaded.
* **Death system** -> If the rocket collides with anything except the spawn point or the finish line, a death animation will be triggered, and the scene will be reloaded (restarting the level).
