# 🎮 Rocket
Get on the rocket and reach the finish line while you are avoiding the danger that's ahead of you. This is a 3D game made with unity.
<img width="1615" height="862" alt="Rocket" src="https://github.com/user-attachments/assets/ead56e77-f503-498c-98d5-dac9ff7fd322" />

## 🛠️ Technical details
* **Movement system** -> made with the Unity's input system: boost(SPACE), rotate left(a)/right(d)
* **Scene manager** -> there are 5 levels, each of them being represented by it's own scene
* **Finish line** -> when the rocket collides with the finish line, there is an animation, and after that, the next scene/level will be loaded
* **Death system** -> if the rocket collides with anything, exept the spawn point or the finish line, an animation will be triggered, and the scene will be reloaded (the level will be restarted)
